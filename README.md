# Forevercraft

A Minecraft datapack for **1.21.11** (Pack Format 94.1) that reimagines vanilla survival with 74+ interconnected systems.

## Features

- **Professions & Mastery** — Weapon, armor, biome, and harvest mastery systems
- **Companions** — Tamed pet system with menus, crates, and devotion tracking
- **Advantage Trees** — Custom skill trees with GUI-based leveling
- **Artifacts** — 12+ healer artifacts with unique abilities and cooldowns
- **World Events** — Dynamic encounters, night terrors, omens, and boss fights
- **Seasons & Calendar** — Time-aware world with visual day/night cycle (slowed 3x)
- **Structures & Loot** — Custom structure crates with refresh timers
- **Cooking & Foraging** — Expanded food and gathering systems
- **Guidestones** — Waypoint teleportation network
- **Constellations & Dreams** — Nighttime stargazing and dream realm mechanics
- **Runeforge & Transmutation** — Crafting and item transformation
- **Journal & Codex** — In-game documentation and lore tracking
- **Trim Effects** — Armor trim patterns grant gameplay bonuses
- **And much more** — Bounties, fishing, luck, moon phases, housing, parties...

## Installation

1. Download or clone this repo
2. Place the `Forevercraft` folder into your world's `datapacks/` directory
3. Run `/reload` in-game

## Requirements

- Minecraft Java Edition **1.21.11**
- Recommended with the **Toodle Pack Prime 2.0** resource pack for custom models and textures
